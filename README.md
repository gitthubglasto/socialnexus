# SocialNexus

Unified social inbox iOS prototype for monitoring conversations, engagement, and trend signals across multiple platforms.

## Included in this scaffold
- Product + UX specification: `Docs/ProductSpecification.md`
- SwiftUI starter architecture in `SocialNexus/`
  - Normalized core models
  - Mock repository/data
  - ViewModels for feed and detail
  - Unified Feed and Post Detail views
  - Tab-based app shell for Feed, Trends, Search, Accounts
